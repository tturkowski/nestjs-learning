# NestJS Learning

## Description

Rest API using NestJS, Prisma based on tutorial @ https://www.youtube.com/watch?v=GHTA143_b-s

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod

# run dev database
npm run db:test:restart
```

## Test

```bash
# run test database
npm run db:test:restart

# e2e tests
npm run test:e2e

# e2e tests (with watch)
$ npm run test:e2e:watch
```
